# AGENTS.md - AI Agent 작업 규칙

## 프로젝트 목적

HTML/CSS/JavaScript 기반 감성 분석 웹 서비스를 구현한다. 사용자가 입력한
텍스트를 OpenAI API로 분석하고 긍정/부정/중립 결과와 부가 정보를
제공한다.

## 작업 원칙

-   요구사항에 없는 기능을 임의로 추가하지 않는다.
-   모든 구현은 PRD와 docs 문서를 기준으로 진행한다.
-   보안상 API Key는 Front-end 코드에 노출하지 않는다.
-   OpenAI 호출은 Node.js Backend에서만 처리한다.
-   UI는 참고 이미지의 감성을 유지한다.

## UI 방향

-   검은 배경
-   중앙 집중형 레이아웃
-   어두운 분위기의 감성 디자인
-   부드러운 Glow 효과
-   흰색 중심 텍스트
-   간결한 입력 경험

## 구현 금지 범위

-   회원가입 기능
-   결제 기능
-   관리자 페이지
-   실시간 채팅
-   복잡한 통계 대시보드

## 완료 기준

-   사용자가 텍스트 입력 가능
-   분석 버튼 동작
-   OpenAI 요청 성공
-   결과 화면 표시
-   오류 상황 안내
-   Supabase 저장 가능

## 검증 기준

1.  빈 입력 처리 확인
2.  정상 문장 분석 확인
3.  API 오류 메시지 확인
4.  모바일 화면 확인
